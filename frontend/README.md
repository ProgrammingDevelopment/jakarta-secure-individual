
# Jakarta Secure Individual - Frontend

Modern React-based security analysis platform.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/ProgrammingDevelopment/jakarta-secure-frontend)

### Environment Variables

Set these in Vercel Dashboard:
```
VITE_API_URL=https://your-backend-url.com
```

### Local Development

```bash
npm install
npm run dev
```

### Build for Production

```bash
npm run build
```
